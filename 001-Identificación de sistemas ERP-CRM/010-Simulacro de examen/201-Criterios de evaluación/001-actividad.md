Trabajo de hoy en clase: Simulacro de examen:
Consulta el modelo de datos que te he enviado al email

Crea un sistema de gestión empresarial. Puedes utilizar IA como apoyo
El sistema estará basado en el stack: HTML, CSS, JS (opcional) en el cliente, Python(flask) SQLite en el servidor
Si lo prefieres, puedes cambiar Python por Nodejs o por PHP, SQlite por MySQL o PostgreSQL
Al inicio del examen se te ha proporcionado un modelo de datos de una aplicación. Construye el sistema de gestión empresarial con ese modelo de datos
En primer lugar asegúrate que funciona, a continuación rellena la rúbrica
El examen, como todas las actividades, se realiza en eval siguiendo la rúbrica de evaluación
